# PI-SegPeriodo 

Interação Dentista/Protético

Plataforma web para interação do dentista com o protético para fazer pedidos de trabalho.

Grupo:
- João Antonio.
- João Afrânio.

======


### 1 - Motivação

Notamos que muitos dentistas utilizam de meios de comunicação para gerenciar seus pedidos ao protético. Nossa intenção 
informatizar esses dados de forma que o usuário possa acessá-los de forma fácil, ágil e organizada.

Com a presença do website o tempo utilizado nessa comunicação será reduzido,pois não será necessário entrar em contato diretamento com o 
protético empregado para : Fazer alterações, confirmar pedidos , fazer novos pedidos.


### 2 - Cronograma


- Setembro: Paginas de Login/Cadastro.

- Outubro: Pagina do Dentista: Fazer pedidos,filtrar proteticos por região.

- Novembro: Pagina do Protetico: Aceitar pedidos,listar trabalhos.
